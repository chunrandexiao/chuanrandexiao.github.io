SSH

cat ~/.ssh/id_rsa.pub 查看本地ssh
ssh-keygen -t rsa -b 4096 -C “your_email@example.com“ 生成本地ssh 敲三次回车

linux 命令

ls 产看文件
ls -a 查看包括隐藏文件
touch 文件 ： 创建文件
mkdir 文件夹 ： 创建文件夹
cd 文件目录 ：进入文件目录
rm -rf 文件 ： 删除文件

提交仓库

git add -A
git commit -m “本次提交的修改的备注”
git commit -am “本次提交的修改的备注”只修改文件，合并-A和-m
git push origin master git 第一次提交到本分支
git push第二次提交到本分支
git pull 抓取线上和远程仓库所有内容

git绑定

git init 初始化
git remote add origin git@github.com:chunrandexiao/youkeda.git 绑定其中的origin可以更换
get remote remove origin解除绑定

hexo

##hexo创建
hexo init blog
cd blog
npm install hexo-deployer-git –save
进入_config.yml
theme: landscape
deploy:
type: git
repository: 你的GitHub仓库地址
branch: master
hexo clean
hexo generate
hexo deploy

新建博客

hexo new 博客名 闯进了新的博客放在source/_posts文件夹内

更换主题

git clone –depth 1 git@github.com:iissnan/hexo-theme-next.git themes/next
在_config.yml中修改theme

分支

创建并切换分支git checkout -b dev
创建新的分支git branch dev
切换分支git checkout dev
分支过滤public
打开。git.ignore确保有public/

本地环境配置

https://git-scm.com/downloads
https://nodejs.org/zh-cn/
npm install -g hexo-cli： 安装hexo
git clone -b dev 仓库地址：到本地
npm install：安装node依赖
